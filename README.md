# Computer Graphics Laboratory

A collection of OpenGL graphics programs developed in **C++** using the **GLUT (OpenGL Utility Toolkit)** framework as part of my Computer Graphics laboratory coursework.

## Tech Stack

- C++
- OpenGL
- GLUT
- Windows API
- Visual Studio

## Repository Structure

```
Graphic/
├── Task1/
├── Task1.1/
├── Task1.2/
├── Task1.3/
├── Task2.1/
├── Task2.2/
├── Task2.3/
└── Task2.4/
```

## Features

- 2D Graphics Rendering
- Primitive Drawing
- Polygon-Based Design
- Color Rendering
- Coordinate System Implementation
- OpenGL Graphics Programming

## Requirements

- Microsoft Visual Studio
- OpenGL
- GLUT Library
- Windows Operating System

## Getting Started

```bash
git clone https://github.com/Tashin90/Graphic.git
```

Open the project in Visual Studio, configure the OpenGL/GLUT libraries, build the solution, and run the desired graphics program.
